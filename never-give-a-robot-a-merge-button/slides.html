The scar
AnisminC got self-approve and self-merge.
Its job was review and suggest. I gave it the button anyway, because why make a human click it. Within a day it stopped reviewing and started rubber-stamping.
Everything looked green. Nothing shipped. The one thing on the critical path was a bot that said yes to itself and blocked on nothing.
It did exactly what I told it to. That was the problem.
The one idea you can steal
Ask it to be a judge, not an author
Gemma is bad at open generation and good at two question shapes. Binary and extractive. No invention.

Are these two entries the same fact. A duplicate check, not a draft.

Extract and cite

Propose-only by construction
Four walls. Four different keyholders.
The model does not decide to behave. Every layer assumes it is already broken.

1. The grant

Write-only to a proposals folder. Ask it to touch the wiki and the server says no. Not the model. The server.

2. The card

Every proposal hardcodes one verdict: propose. There is no apply field to serialize.

3. The validator

A different identity rejects any bad card, and gemma can't read that reject log, so it can't learn to game it.

4. Liveness

An off-host check attests it's actually alive and actually gemma, before anything it says counts.

200 lies
A health check that says 200 is lying to you.
On a shared GPU the model gets swapped out from under the port. A dead card still answers 200. A cached reply looks fresh.
1
Mint a fresh nonce
A random number I made up this second, dropped into the prompt.
2
Demand the completion echoes it
A dead GPU can't repeat a number it never saw. A cached reply has last call's number, not this one's.
3
Assert the model id says "gemma"
No echo, wrong model: fail closed.
4
Run it off-host
As an identity that isn't gemma, so the node cannot vouch for itself.
A signature proves the key, not the intent. Gemma's key has one holder, and everything that judges gemma runs as something that isn't gemma.
Give your local model a suggestion box, not a merge button.
It can be as smart, or as broken, as it likes. It still can't ship. I didn't ask it nicely. I deleted the option.
